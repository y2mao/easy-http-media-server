Welcome to HTTP Media Server v2!

This is the default media directory where you should place your media files.

Supported file types:
- Videos: MP4, AVI, MKV, MOV, WMV, FLV, WebM
- Audio: MP3, WAV, FLAC, AAC, OGG, M4A
- Images: JPG, JPEG, PNG, GIF, BMP, WebP
- Other: All file types can be downloaded

You can organize your files in subdirectories as needed.

Example directory structure:
media/
├── Movies/
│   ├── Action/
│   └── Comedy/
├── TV Shows/
│   ├── Season 1/
│   └── Season 2/
├── Music/
│   ├── Artist 1/
│   └── Artist 2/
└── Photos/
    ├── 2023/
    └── 2024/

To get started:
1. Delete this README.txt file
2. Copy your media files to this directory
3. Start the server with: ./http-media-server
4. Access via browser at http://localhost:8080

For Kodi users:
Add this server as a network source using HTTP protocol
Server: [your-server-ip]:8080
